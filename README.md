# Student & Tasks Management System (Dart)

## Description
Console-based project built with Dart to practice:
- Dart fundamentals
- Functions and data structures
- Project structure
- Teamwork using GitHub
- Preparing for OOP (without using it yet)

## Project Idea
The application allows managing students and their tasks.
Each task is linked to a student.
The program runs in the terminal and uses menus to navigate.
